pytlas
======

Python 3 assistant made to be super simple to setup!

Installation
------------

pip
~~~

.. code-block:: bash

  $ pip install pytlas

source
~~~~~~

.. code-block:: bash

  $ git clone https://github.com/atlassistant/pytlas.git
  $ cd pytlas
  $ python setup.py install

  or

  $ pip install -e .

Testing
-------

.. code-block:: bash

  $ cd tests/
  $ python -m unittest -v
